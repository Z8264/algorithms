# 栈 (Stack)



| 方法     | 描述                           |
| -------- | ------------------------------ |
| push     | 添加一个元素                   |
| pop      | 移除栈顶的元素                 |
| peek     | 返回栈顶的元素，该元素不被移除 |
| isEmpty  | 栈是否为空                     |
| clear    | 清空栈里的元素                 |
| size     | 返回栈里的元素个数             |
| toArray  | 转换为数组                     |
| toString | 转换为字符串                   |
