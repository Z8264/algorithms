# 队列 (Queue)




| 方法     | 描述                                 |
| -------- | ------------------------------------ |
| push     | 向队列尾部添加一个元素               |
| shift    | 移除队列的第一个元素                 |
| peek     | 返回队列的第一个元素，该元素不被移除 |
| isEmpty  | 队列是否为空                         |
| clear    | 清空队列里的元素                     |
| size     | 返回队列里的元素个数                 |
| toArray  | 转换为数组                           |
| toString | 转换为字符串                         |
